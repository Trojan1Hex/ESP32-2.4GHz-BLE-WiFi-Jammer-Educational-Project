🔧 Hardware Notes

Neopixel LED is optional — the system works without it.

For better stability:
Use a 10V / 100µF capacitor

Connect it in parallel between 3.3V and GND of each NRF module


📥 Setup Instructions

Download the ZIP file of the project

Extract (unzip) the main ZIP file

Inside it, extract the “code” folder (if compressed)

Open the project file (.ino) in Arduino IDE

Go to:

Sketch → Include Library → Add .ZIP Library

Add all required libraries from the project folder

Select your board:

ESP32 Dev Module (or your specific ESP32 board)

Select correct COM port

Click Upload

✅ After Upload

Power the ESP32

The system will start automatically

Your device is now ready to operate
